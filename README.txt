In order to compile:

* Install ruby (or iron ruby) and add it to the path
* Install nuget and add it to the path
* Run: rake (should build)
* Run: rake test (should run all the tests)
* Run: rake test:acceptance (should run all features)

Description:
The demos is a ASP.NET MVC 3 application that uses the REST API to show the presenters, sessions and tracks.
